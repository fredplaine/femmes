### Mode Filage ⏯
Accessible via le bouton **"⏯ Mode Filage"** à droite du compteur de ligne en dessous des cadres de scenes (A1, A2, ... D6, FIN) / AvaNT de la boite Mode Turbo

Simule un vrai filage debout, sans avoir à cliquer "Suivant" :
- **Répliques des autres** → texte affiché + lues à voix haute
- **Vos répliques (Le President)** → texte masqué ("🎭 À vous de jouer !") + silence chronométré à la même vitesse, pour vous donner exactement le temps de les dire. Applique la vitesse parametree. 
- Avance automatiquement de réplique en réplique


Paramètres du Mode Filage (panneau visible quand actif) :
- **Vitesse** : 0.5× à 2× (multiplicateur sur la vitesse générale)
- **Auto-avancer** : activable/désactivable
